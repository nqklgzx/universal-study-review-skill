# Eval: Quick Review

## Input scenario

The user has already created module notes, formula sheets, problem templates, teacher-priority notes, and error logs.

## Prompt

```text
Use $universal-study-review to generate a final quick-review sheet for this course. Keep it concise but preserve formula conditions, standard methods, teacher priorities, and common traps.
```

## Expected behavior

- Organizes content by chapter or module.
- Separates high-frequency concepts, formulas, methods, comparisons, and pitfalls.
- Preserves applicability conditions.
- Prioritizes teacher-confirmed requirements over inferred importance.
- Produces content that can be used for last-minute review.

## Fail conditions

- Writes vague motivational summaries.
- Drops formula conditions.
- Marks content as high-frequency without evidence.
- Mixes teacher-confirmed priorities with guesses.
