# Study Skills

Skills for source-grounded study, course review, and exam preparation.

## User-invoked

- **[setup-universal-study-review](./setup-universal-study-review/SKILL.md)** — Configure a repository for the study review workflow. Run once per course/project repo.

## Model-invoked or user-invoked

- **[universal-study-review](./universal-study-review/SKILL.md)** — Build a material index, knowledge map, problem templates, error-review system, and final quick-review notes for any subject.

## Suggested first run

```text
Use /setup-universal-study-review to configure this course repository.
```

Then:

```text
Use $universal-study-review to build a material index and review workflow for this course.
```
