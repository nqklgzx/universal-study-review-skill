# Universal Study Review

Turn any subject's materials into a reusable learning system.

## Core workflow

1. Inspect available materials before summarizing.
2. Build a material index.
3. Build a course/module knowledge map.
4. Extract definitions, formulas, methods, and conditions.
5. Integrate teacher notes, exam rules, and priorities.
6. Create problem-type templates.
7. Solve examples with exam-style reasoning.
8. Maintain an error log.
9. Produce final quick-review materials.

## Rules

- Do not invent page numbers, file structures, formulas, teacher requirements, or problem conditions.
- Use provided materials as the source of truth.
- If evidence is missing, state what is missing.
- Prefer reusable Markdown outputs with clear headings.
- For problems, include method selection, derivation, final result, and mistakes to avoid.
