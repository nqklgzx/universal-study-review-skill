# Examples

This directory contains lightweight examples showing how to apply the Universal Study Review Skill to different subjects.

## Included examples

- `signals-and-systems.md` — engineering/math-heavy course with formulas, transforms, problem solving, and exam quick review.
- `digital-logic.md` — circuit/design course with truth tables, standard solution workflows, and common traps.
- `ai-math-foundations.md` — math foundations course with concepts, proofs, formulas, and standard problem templates.

Each example shows:

1. suggested input materials;
2. first prompt to run;
3. expected outputs;
4. recommended file names;
5. quality checks.
