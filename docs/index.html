Project Overview

这是什么项目

Universal Study Review Skill 是一套通用的 AI 复习工作流。它不是单一提示词，而是一组可部署到 GitHub、Codex、Markdown 软件和网页中的复习流程模板。

核心目标是：先建立资料索引，再重构课程知识脉络，再融合老师重点和题型训练，最后生成考前速看材料。后续所有问答都先定位资料来源，再进行解释或解题。

资料查阅清单 章节知识脉络 公式与方法提取 题型库 错题复盘 考前速看 Codex Skill GitHub Pages

Why

它解决什么问题

资料多但不可查

PPT、教材、笔记、题库堆在一起时，先生成资料索引，保留文件结构、章节定位和使用规则。

总结太概括

要求按章节、页码、公式、条件、题型和易错点整理，不用空泛概括替代具体内容。

AI 容易脱离资料

后续问答默认先查阅资料清单，再定位原文件，无法验证时明确说明缺失信息。

做题过程不标准

题目解答按“考点定位—读题分析—解题思路—具体推导—最终答案—易错点”输出。

Workflow

标准复习工作流

S00设定复习助手角色
S01遍历资料并建立查阅清单
S02建立全课程知识脉络
S03按章节或模块整理资料结构
S04提取公式、定义、结论与方法
S05融合课堂重点、录音、教师要求
S06建立题型库与标准解题模板
S07深入讲解单个知识点
S08标准化解答具体题目
S09整理错题、易错点和辨析点
S10生成考前速看资料
S11后续问答时按资料定位回答
S12迭代更新资料库和提示词

Outputs

最终会得到哪些复习产物

资料查阅清单

列出每个文件的内容、结构、章节定位、可查阅知识点和后续使用规则。

章节/模块脉络

按资料原顺序整理每部分讲什么、公式条件、题型对应和重要程度。

题型库与模板

把知识点转化为标准题型，形成可复用的解题步骤和检查方法。

考前速看资料

压缩为高频考点、核心公式、适用条件、标准方法和最后易错点。

Install

安装与使用

安装命令放在这里，作为项目内容之后的使用入口。

方式一：skills 安装器

npx skills@latest add nqklgzx/universal-study-review-skill

方式二：Windows PowerShell 远程安装到当前仓库

powershell -NoProfile -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/nqklgzx/universal-study-review-skill/main/scripts/bootstrap-codex-skill.ps1 | iex"

方式三：本地克隆后安装到目标仓库

powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\install-codex-skill.ps1 -TargetRepo "D:\Your\CourseRepo" -Force -UpdateAgents

Codex 调用

使用 $universal-study-review，把这门课的资料整理成复习查阅清单、章节脉络、题型模板和考前速看资料。
